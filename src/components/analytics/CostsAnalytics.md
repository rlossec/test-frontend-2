# CostsAnalytics - Analyse et Documentation

## 1. Consignes

Le composant `CostsAnalytics` affiche l'analyse des coûts des outils internes de l'organisation. Il doit présenter :

- L'évolution mensuelle des dépenses
- La répartition des coûts par département
- Les outils les plus coûteux
- Le suivi du budget mensuel
